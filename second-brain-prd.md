## 1. 产品概述

### 1.1 产品定位
**第二大脑 (Second Brain)** 是一款面向AI从业者/创业者的个人知识管理系统，旨在帮助用户集中管理记忆、文档、任务和Agent工作状态。

### 1.2 目标用户
- AI创业者
- 内容创作者
- 需要管理多个自动化Agent的用户

### 1.3 核心价值
- **一体化**：记忆、文档、任务、Agent状态集中管理
- **可视化**：通过仪表盘直观了解系统运行状态
- **可追溯**：支持按时间筛选和搜索历史数据
- **实时性**：数据与Cron任务执行时间同步

## 5. 技术方案

### 5.1 技术栈
- **前端**: Next.js 15 + React + TypeScript
- **样式**: Tailwind CSS
- **数据库**: Supabase (PostgreSQL)
- **部署**: Vercel

### 5.2 数据表设计

```sql
-- memories 表
CREATE TABLE memories (
  id UUID PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT,
  date DATE NOT NULL,
  type TEXT CHECK (type IN ('daily', 'long-term', 'evolution')),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP
);

-- documents 表
CREATE TABLE documents (
  id UUID PRIMARY KEY,
  title TEXT NOT NULL,
  path TEXT NOT NULL,
  type TEXT,
  date DATE NOT NULL,
  size INTEGER,
  created_at TIMESTAMP DEFAULT NOW()
);

-- tasks 表
CREATE TABLE tasks (
  id UUID PRIMARY KEY,
  name TEXT NOT NULL,
  schedule TEXT,
  status TEXT CHECK (status IN ('ok', 'error', 'running')),
  last_run TIMESTAMP,
  last_duration INTEGER,
  next_run TIMESTAMP,
  error_count INTEGER DEFAULT 0
);
```

### 5.3 API设计

| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | /api/memories | 获取记忆列表 |
| POST | /api/memories | 创建记忆 |
| GET | /api/documents | 获取文档列表 |
| GET | /api/tasks | 获取任务列表 |
