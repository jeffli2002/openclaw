## 1. 产品概述

### 1.1 产品定位
**第二大脑 (Second Brain)** 是一款面向AI从业者/创业者的个人知识管理系统，旨在帮助用户集中管理记忆、文档、任务和Agent工作状态。

### 1.2 目标用户
- AI创业者
- 内容创作者
- 需要管理多个自动化Agent的用户

### 1.3 核心价值
- **一体化**：记忆、文档、任务、Agent状态集中管理
- **可视化**：通过仪表盘直观了解系统运行状态
- **可追溯**：支持按时间筛选和搜索历史数据
- **实时性**：数据与Cron任务执行时间同步

### 5.2 数据表设计

```sql
-- memories 表
CREATE TABLE memories (
  id UUID PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT,
  date DATE NOT NULL,
  type TEXT CHECK (type IN ('daily', 'long-term', 'evolution')),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP
);

-- documents 表
CREATE TABLE documents (
  id UUID PRIMARY KEY,
  title TEXT NOT NULL,
  path TEXT NOT NULL,
  type TEXT,
  date DATE NOT NULL,
  size INTEGER,
  created_at TIMESTAMP DEFAULT NOW()
);

-- tasks 表
CREATE TABLE tasks (
  id UUID PRIMARY KEY,
  name TEXT NOT NULL,
  schedule TEXT,
  status TEXT CHECK (status IN ('ok', 'error', 'running')),
  last_run TIMESTAMP,
  last_duration INTEGER,
  next_run TIMESTAMP,
  error_count INTEGER DEFAULT 0
);
```

```typescript
interface OfficeState {

  id: string;

}

interface ActivityLog {
  id: string;
  timestamp: Date;
  agentId: string;
  agentName: string;
  message: string;
  type: 'info' | 'success' | 'warning' | 'error';
}
```

### 7.7 技术实现建议

#### 7.7.1 2D 渲染方案

**推荐方案**: 使用 React + CSS/SVG 实现
- **优势**: 轻量、响应式、易于维护
- **替代**: Canvas / Konva.js (如需复杂交互)

#### 7.7.2 人物图片资源

1. **AI 生成**: 使用 DALL-E/Midjourney 生成统一风格图片
2. **图库购买**: Envato Elements / Creative Market
3. **自定义**: 使用 Avataaars / ProfilePicture 等工具

#### 7.7.3 实时更新

- **方案 A**: 轮询 (Polling) - 每 5 秒请求一次
- **方案 B**: Server-Sent Events (SSE) - 实时推送
- **推荐**: 方案 A (简单可靠)
