# SEO Keyword Research Skill

## Overview

This skill helps AI agents conduct keyword research and create SEO-optimized content for websites and landing pages.

## When to Use

- When you need to research keywords for a website
- When building SEO-optimized landing pages
- When analyzing search trends and competition
- When creating content strategy

## Keyword Research Process

### 1. Seed Keywords

Start with core keywords related to your business:
- Product/service category names
- Problem-solution terms
- Competitor names

### 2. Keyword Metrics

For each keyword, analyze:
- **Search Volume**: Monthly searches (target >1000 for commercial keywords)
- **KD (Keyword Difficulty)**: Competition level (target KD <30 for new sites)
- **Intent**: Informational, navigational, transactional
- **CPC**: Commercial value indicator

### 3. Long-tail Opportunities

Find long-tail keywords (3-5 words):
- Lower competition
- Higher conversion intent
- Question-based queries

## SEO Content Structure

### Landing Page Elements

1. **Title Tag** (50-60 characters)
   - Include primary keyword
   - Compelling action words

2. **Meta Description** (150-160 characters)
   - Summary of page content
   - Include CTA

3. **H1 Heading**
   - Clear keyword focus
   - Answer user's intent

4. **H2-H6 Structure**
   - Logical content hierarchy
   - Include related keywords

5. **Content Length**
   - Minimum 300 words for thin pages
   - 1500+ words for pillar content

6. **Internal Linking**
   - Link to related content
   - Use descriptive anchor text

## Example: SEO Page Template

```markdown
# [Primary Keyword] - [Benefit/Result]

## Introduction
[Hook + Problem statement + Solution preview]

## [Section 1: Keyword]
[Value proposition + supporting details]

## [Section 2: Keyword]
[Value proposition + supporting details]

## [Section 3: Keyword]
[Value proposition + supporting details]

## CTA
[Clear call to action]
```

## Best Practices

1. **One keyword per page** - Focus on primary keyword
2. **Natural keyword placement** - Don't over-optimize
3. **Mobile-first** - Ensure mobile compatibility
4. **Page speed** - Target <3 second load time
5. **User intent match** - Content matches search intent

## Tools Reference

- **Google Keyword Planner** - Free keyword research
- **Ahrefs/SEMrush** - Advanced analysis
- **Google Search Console** - Performance tracking
- **Google PageSpeed Insights** - Performance testing

## Output Format

When creating SEO content, provide:

1. **Keyword List** (CSV format)
   - Keyword, Volume, KD, Intent, CPC

2. **Content Brief**
   - Target keyword
   - Word count
   - Required sections
   - Internal linking suggestions

3. **Meta Tags**
   - Title tag
   - Meta description
   - OG tags
