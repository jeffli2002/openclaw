## 课程体系

### 一、一日课程（标准版）

#### 模块 A：OpenClaw 基础（2小时）
1. OpenClaw 是什么？产品定位与核心理念
2. OpenClaw vs 普通 AI Agent 的本质区别
3. OpenClaw 架构解析（Gateway、Agents、Skills）
4. 国内 OpenClaw 相关产品差异化对比

#### 模块 B：OpenClaw 部署与安装（2小时）
1. 环境准备（Node.js、Docker、云服务器）
2. 本地部署实战
3. 云端部署（VPS/云服务器）
4. 飞书/微信/Telegram 多渠道配置
5. 常见部署问题与解决方案

#### 模块 C：OpenClaw 商业应用场景（2小时）
1. **办公场景**：日程管理、信息汇总、任务分发
2. **自媒体场景**：内容创作自动化、社交媒体运营
3. **客服场景**：智能客服、工单处理
4. **销售场景**：获客、转化、CRM 自动化

#### 模块 D：实战工作坊（2小时）
1. 部署自己的第一个 Agent
2. 配置第一个自动化工作流
3. 场景实战：搭建一个小型客服/内容系统
