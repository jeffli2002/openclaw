# Second Brain Office - Agent 协作功能 PRD

## 更新日期
2026-03-17

## 功能概述
在 Office 页面中，当多个 Agent 进行任务协作/沟通时，显示他们在会议室中进行面对面沟通的动画。

## 触发条件
多个 Agent 之间有任务协作/传递时

## 场景规则

| 参与 Agent 数 | 会议区域 |
|-------------|---------|
| 2个 | 小会议室 (Meeting A) |
| ≥3个 | 大会议室 (Meeting B) |

## 判定逻辑
1. 检测到某任务涉及多个 Agent（如 Chief 分发给 Content）
2. 相关 Agent 从工位走到对应会议区域
3. 动画显示"沟通中"状态
4. 沟通完成后各回工位

## 会议区域坐标（参考）
- Meeting A (小): 约 (654, 214)
- Meeting B (大): 约 (1078, 214)

## 状态优先级
1. 协作/沟通中 → 会议室
2. running/busy → 工位工作
3. idle > 1小时 → 休息区坐着
4. idle → 走动

## 技术实现
- 检测 agent status 变化
- 计算参与 Agent 数量
- 路由到对应会议室
- 动画平滑过渡
