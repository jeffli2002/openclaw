## 🔥 全球AI

**[Anthropic成立 Anthropic Institute，把“前沿AI的社会冲击”正式机构化](https://www.anthropic.com/news/the-anthropic-institute)**
发布时间：03-11 17:50（北京时间）
Anthropic 宣布成立 The Anthropic Institute，专门研究强 AI 对就业、经济、法律和社会治理的影响，并由 Jack Clark 牵头。**这意味着“模型公司顺手做政策沟通”的阶段正在过去，头部实验室开始把治理研究变成常设组织。** 对行业来说，接下来关于 AI 价值对齐、法律系统适配和预警机制的讨论，可能会从抽象辩论转成更制度化的产出。

**[OpenAI发布一篇很实用的安全文章：怎么让 AI Agent 更抗 Prompt Injection](https://openai.com/index/designing-agents-to-resist-prompt-injection)**
发布时间：03-11 19:30（北京时间）
OpenAI 的核心判断很明确：现实世界里的 prompt injection，越来越像社会工程攻击，而不只是“坏字符串”。**真正有效的防守重点，不只是过滤输入，而是把 agent 的权限、执行边界和失败后影响范围收紧。** 这会直接影响后面一波 Agent 产品的系统设计：谁的权限模型更稳，谁才更有资格进企业生产环境。

**[OpenAI继续补 Agent 基础设施：把 Responses API 真正配上“电脑环境”](https://openai.com/index/equip-responses-api-computer-environment)**
发布时间：03-11 19:00（北京时间）
这篇工程文披露了 OpenAI 如何把 shell、容器工作区、文件系统、受限网络和 agent loop 串起来。**信号很强：行业竞争正在从“模型会不会答题”，转向“平台能不能稳定跑真实工作流”。** 对开发者来说，这类托管执行环境一旦成熟，会明显降低自建 agent runtime 的门槛。

## 📝 虾仔观点
