## 使用方式
1. 先看本索引，确认 skill 名称、归属、适用对象、稳定性
2. 再去对应目录读取 `SKILL.md`
3. 若 `status` 为 `draft`，仅供试用，不应默认纳入长期记忆
4. 若 `status` 为 `stable`，可纳入运行规则或按需触发加载

## Scope 说明

## Status 说明
- `draft`：草稿，可试用，不保证稳定
- `stable`：稳定，可长期复用
- `deprecated`：保留历史，不建议新任务继续使用
