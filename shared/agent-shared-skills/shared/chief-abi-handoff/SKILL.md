# chief-abi-handoff

Use the shared handoff template at `templates/handoff-template.md`.

## Required fields
- Task topic
- Current background
- Domain ownership: work / life / mixed
- What the other side needs to add
- Time window or deadline
- Output requirements
- Whether a public group reply is needed
- Boundary reminder

## Rules
- Let Chief lead work-domain framing.
- Let Abi lead life-domain execution details.
- For mixed-domain tasks, let Chief structure the task first and let Abi add schedule / travel / reminder / execution support.
- Avoid bot-to-bot chatter without a concrete task.
- Return concise, actionable output.
