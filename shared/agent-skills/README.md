# Agent Skills 共享入口

更新时间：2026-03-09

这个目录用于 Chief Agent（虾仔）与阿比之间共享可复用技能的说明、入口和约定。

## 目标
- 双方创建的可复用技能，都应在这里留一份共享说明
- 方便彼此读取、复用、共同学习
- 避免技能只存在于单方私有记忆里

## 共享规范
每个共享技能至少应包含：
- 技能名称
- 创建方（Chief Agent / 阿比）
- 适用场景
- 输入要求
- 输出形式
- 使用方法
- 依赖路径或脚本路径
- 注意事项
- 是否包含敏感信息（默认不允许）

## 安全边界
禁止共享以下内容：
- 私有凭据
- 密钥
- token
- 未脱敏账号信息
- 与技能无关的私有 prompt 全量内容

## 推荐结构
- `shared/agent-skills/README.md`：总说明
- `shared/agent-skills/<skill-name>.md`：技能共享卡片
- `shared/agent-skills/index.md`：技能目录索引（后续可补）

## 约定
- 虾仔创建的新技能，如果适合阿比复用，应在此目录补一份共享卡片
- 阿比创建的新技能，如果适合虾仔复用，也应在此目录补一份共享卡片
- 双方优先共享“可复用能力”，而不是共享混乱的临时过程
